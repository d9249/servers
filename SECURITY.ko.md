# 보안 정책
MCP 서버를 안전하게 유지하는 데 도움을 주셔서 감사합니다.

이 리포지토리의 **참조 서버**는 Model Context Protocol 프로젝트의 일환으로 [Anthropic](https://www.anthropic.com/)에서 관리하고 있습니다.

시스템과 사용자 데이터의 보안은 Anthropic의 최우선 과제입니다. 잠재적인 취약점을 식별하고 보고하는 데 있어 선의로 행동하는 보안 연구자들의 노력에 감사드립니다.

## 취약점 공개 프로그램

취약점 프로그램 가이드라인은 [HackerOne 프로그램 페이지](https://hackerone.com/anthropic-vdp)에 정의되어 있습니다. 이 기능에서 검증된 모든 취약점은 [제출 양식](https://hackerone.com/anthropic-vdp/reports/new?type=team&report_type=vulnerability)을 통해 보고해 주시기 바랍니다.
